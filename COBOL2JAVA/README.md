# Legacy COBOL Modernization to Java

A modernization lab using a COBOL Financial Calculator (COBCALC, COBLOAN, COBVALU) from IBM's z/OS sample programs to show how IBM Bob understands legacy mainframe idioms and converts them to clean, object-oriented Java. You generate the Java port, a JUnit test suite, a modern UI, a REST API wrapper, and architecture documentation for a safe, incremental modernization path. (~30–45 min)
